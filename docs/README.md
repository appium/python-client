### How to generate doc

```bash
$cd python-client/docs
$bash generate.sh
```

### How to check generated doc

```bash
$cd python-client/docs
$bash generate.sh
$cd python-client/docs/_build/html
$python -m http.server 1234
```

Access to `http://localhost:1234` on web browser
